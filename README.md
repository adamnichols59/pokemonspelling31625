## 🗺️ How to Play

Bella travels across a **Pokémon-style world map**, moving through 5 locations as she works through 20 spelling words. At each stop, a wild Pokémon appears — spell the word correctly to catch it and add it to your Pokédex!

| Location | Words |

| ⛰️ Pewter City | Words 9–12 |

## 🎮 Game Features

- **🔊 Read-aloud** — each word is spoken, used in a sentence, then spoken again
- **💡 Hints** — first wrong answer shows a hint automatically (no penalty)
- **❤️ Hearts** — 3 lives; lose one on the second wrong answer for the same word
- **⚡ Pokémon encounters** — 20 different Pokémon to catch, one per word
- **🎵 Chiptune music & sound effects** — 8-bit background music, encounter sounds, catch sounds, and a victory fanfare
- **🏆 Perfect Score Prize** — spell all 20 words correctly on the first try with no hints to unlock the $1.00 prize screen!

## 🛠️ Tech

- Single HTML file — no build tools, no dependencies to install
- Voice powered by [Puter.js](https://puter.com) — free, no account required for players
- All sound effects and background music generated with the Web Audio API (no audio files needed)
- Works on any device with a modern browser
